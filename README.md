List of most active GitHub users based on `public contributions` `private contributions` and `number of followers`  by country or state. The list updated `2022/4/7 1:41 PM UTC`.

This repository contains users `59 countries` and `319 cities`. 
To get into the list you need to have minimum number of followers that varies in each country. The list can be found in [config.json](https://github.com/halka9000stg/top-github-users/blob/main/config.json).
